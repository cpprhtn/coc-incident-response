### [safer-events-pre-work.pdf](./safer-events-pre-work.pdf)

안전한 이벤트를 위한 사전 작업

1. 장소를 평가할 때:
   1. 장소 교통 접근성
   2. 보안 및 CCTV 여부 파악
   3. 비상구 등의 시설 확인
2. 식사 제공시:
   1. 비건, 할랄 등의 식습관 고려
3. 티켓 판매 개시전:
   1. 행동강령 검토
   2. 티켓을 구매하기 전에 모든 참석자가 행동강령에 동의하도록
4. 자원봉사자 일정을 계획할 때:
   1. 모든 자원봉사자에게 CoC 교육
   2. 사고 대응 담당자와 조용한 방의 위치(혹은 CoC 대응을 위한 장소)등을 안내합니다.
